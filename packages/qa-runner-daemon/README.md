# @talenttic/qa-runner-daemon

Local daemon that serves the QA Runner UI and reads/writes QA suites in your repo.

## Install

```bash
npm install -D @talenttic/qa-runner-daemon
```

## Usage

```bash
npx qa-runner daemon start
```

Then open:

```
http://localhost:4545/ui
```
